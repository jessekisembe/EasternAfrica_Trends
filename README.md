# Contrasting Changes in Rainy Season Length, Rainfall Frequency, and Intensity across Eastern Africa Replication Code

This repository contains replication code for Kisembe et al. (2026), "Contrasting Changes in Rainy Season Length, Rainfall Frequency, and Intensity across Eastern Africa" published in the *Journal of Hydrometeorology*. This includes the code necessary to calculate the trends used in the analysis, in addition to the code necessary to reproduce all main text and supplementary figures.
